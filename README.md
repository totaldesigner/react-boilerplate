# React Boilerplate

## Node.js 다운로드 및 설치
https://nodejs.org/ko/download/

## Yarn 설치

```bash
$ npm install --global yarn
```

## Node 모듈 설치

```bash
$ npm install

# or
$ yarn
```

## 빌드

```bash
$ npm run build

# or
$ yarn build
```

## 실행

```bash
# development 모드로 실행
$ yarn start

# or
$ npm run start

# production 모드로 실행
$ yarn start:prod
```

## 테스트

```bash
$ yarn test

# coverage/Iconv-report/index.html 에서 테스트 커버리지 확인
# html 파일을 브라우저에 휙 던져보자!! (로컬파일 바로 실행)
```
